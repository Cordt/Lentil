# Lentil

## Overview
Lentil is a native iOS Client for Lens Protocol. It does not aim to be a clone of Twitter, YouTube or others, but instead to showcase and explore the full set of functionalitites, as offered by the protocol.
The app is built using 
* [https://github.com/pointfreeco/swift-composable-architecture](The Composable Architecture) as the core architecture of the app
* [https://developer.apple.com/xcode/swiftui/](SwiftUI) for the interface
* [https://github.com/argentlabs/web3.swift](Web3.swift) to handle the wallet integration
